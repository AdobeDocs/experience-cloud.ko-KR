# 샌드박스 선택 {#sandbox}

경험 롤아웃은 Adobe Experience Platform을 기반으로 구축됩니다. 기능 플래그를 사용하여 작업하기 전에 다른 Adobe Experience Platform 애플리케이션에서 하듯이 현재 환경에 해당하는 샌드박스를 선택합니다.

## 샌드박스를 선택하는 방법 {#how-to}

기능 플래그를 만들거나 수정하기 전에 경험 롤아웃 콘솔의 맨 위 탐색 막대에 있는 샌드박스 전환기를 사용하여 올바른 샌드박스를 선택합니다.
