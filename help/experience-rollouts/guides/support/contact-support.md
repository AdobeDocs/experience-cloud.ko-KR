# 지원 문의 {#contact-support}

Experience Rollouts 지원 팀의 지원이 필요한 경우 Adobe의 표준 지원 채널을 사용하여 요청을 제출하십시오. 지원 팀은 들어오는 요청을 모니터링하고 가능한 한 빨리 응답합니다.

## 지원 요청을 제출하는 방법 {#submit-request}

지원 요청을 제출하려면 다음 단계를 수행합니다.

1. [Adobe Admin Console](https://adminconsole.adobe.com)에 로그인합니다.
2. 위쪽 탐색에서 **지원**(으)로 이동합니다.
3. **지원 티켓 열기**&#x200B;를 선택합니다.
4. 문제에 대한 세부 정보를 포함하여 요청 양식을 입력합니다.
5. **경험 롤아웃** 제품에 티켓을 할당하고 제출하십시오.

또는 경험 롤아웃 콘솔의 오른쪽 패널에서 사용할 수 있는 **지원 문의** 옵션을 사용하여 제품에서 바로 지원 요청을 열 수 있습니다.

## 요청에 포함할 내용 {#what-to-include}

지원 팀이 문제를 최대한 효율적으로 조사하도록 지원하려면 지원 요청에 다음 정보를 포함하십시오.

* **환경** — 문제가 스테이징, 프로덕션 또는 둘 다에서 발생하는지 여부를 지정합니다.
* **기능 플래그 또는 릴리스** — 이름, 키 또는 영향을 받는 기능 플래그, 기능 그룹 또는 릴리스에 대한 직접 링크를 제공합니다.
* **사용자 데이터 테스트** — 문제가 발생한 사용자의 사용자 식별자(이메일, GUID 또는 방문자 ID)를 포함합니다.
* **재현 단계** — 문제를 일으킨 정확한 단계를 설명합니다.
* **예상과 실제 동작 비교** - 예상되는 동작과 발생한 동작을 설명합니다.
* **스크린샷 또는 오류 메시지** — 문제를 설명하는 스크린샷, 오류 메시지 또는 API 응답을 첨부합니다.
* **다른 관련 컨텍스트** - SDK 버전, 응용 프로그램 클라이언트 ID 또는 관련된 최근 구성 변경 내용이 포함됩니다.

자세한 정보를 제공할수록 지원 팀이 문제를 신속하게 진단하고 해결할 수 있습니다.
