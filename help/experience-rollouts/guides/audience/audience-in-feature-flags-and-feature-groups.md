# 기능 플래그 및 기능 그룹의 대상자 {#audience-overview}

경험 롤아웃은 기능 플래그 및 기능 그룹에 대한 풍부한 대상 타깃팅을 제공합니다. 프로필 속성, 컨텍스트 변수 및 대상 세그먼트를 결합하여 기능을 수신할 사용자를 정확하게 제어할 수 있습니다.

## 프로필 규칙 {#profile-rules}

프로필 규칙을 사용하면 국가, 이메일 도메인, 계정 유형, 사용자 ID 등과 같은 속성에 따라 사용자를 타깃팅할 수 있습니다. AND/OR 연산자를 사용하여 여러 속성을 결합하고 복잡한 타깃팅 시나리오에 대한 중첩 논리를 빌드할 수 있습니다.

프로필 규칙을 추가하려면 기능 플래그 또는 기능 그룹의 **대상자** 탭으로 이동하여 **프로필** 아래에 조건을 추가하십시오.

대상 규칙 집합을 재사용 가능한 **대상**(으)로 저장하여 여러 플래그 및 그룹에서 사용할 수 있습니다.

컨텍스트 기반 타깃팅(예: 사용자의 활성 언어 또는 클라이언트 유형별 타깃팅)에 대해서는 [대상 규칙에 컨텍스트 사용](using-context-in-audience-rules.md)을 참조하십시오.

## 세그먼트 {#segments}

기능 플래그 및 기능 그룹은 대상 세그먼트를 지원합니다. 세그먼트를 사용하여 다음을 수행할 수 있습니다.

* 여러 기능 플래그 및 그룹에 대해 사전 정의된 단일 대상 정의 사용
* 대상 타깃팅에 **이벤트 기반 기준**&#x200B;을 포함합니다. 프로필 규칙만으로는 가능하지 않습니다.

세그먼트를 사용하려면 **대상자** 탭으로 이동하여 목록에서 하나 이상의 세그먼트를 선택하십시오. AND/OR 연산자를 사용하여 여러 세그먼트를 결합하고 프로필 또는 컨텍스트 필터를 추가할 수 있습니다.

>[!NOTE]
>
>대상 세그먼트를 만들려면 **Test Manager** 역할 이상이 필요합니다.

## 대상 계산기 {#audience-calculator}

대상 기준을 정의한 후 맨 아래 막대에서 **계산**&#x200B;을 선택하여 자격이 있는 사용자의 예상 수를 얻습니다. 이렇게 하면 기능을 활성화하기 전에 타겟팅에 도달했는지 확인하는 데 도움이 됩니다.

>[!NOTE]
>
>컨텍스트 값은 런타임 시 결정되며 미리 예측할 수 없기 때문에 컨텍스트 변수가 기준에 포함될 때 대상 계산기는 값을 반환하지 않습니다.
