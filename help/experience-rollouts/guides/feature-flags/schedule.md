# 예약 {#schedule}

예약은 기능 플래그 또는 기능 그룹이 자동으로 활성화되도록 미래의 날짜 및 시간을 설정할 수 있는 선택적 기능입니다. 수동 켜기/끄기 전환은 계속 사용할 수 있으며 예약으로 바꿀 필요가 없습니다.

다음에 대해 일정 조정이 지원됩니다.

* 기능 플래그
* 기능 그룹

## 1단계: 일정 설정 {#set-schedule}

1. 콘솔에서 기능 플래그 또는 기능 그룹을 엽니다.
2. 화면 오른쪽에 있는 **예약** 단추를 선택합니다.
3. 팝업에서 **시작 날짜 및 시간**&#x200B;을 설정하고 **시간대**&#x200B;를 선택합니다.

## 2단계: 저장 {#save}

**일정 설정**&#x200B;을 선택한 다음 설정을 저장합니다. 저장할 때까지 일정이 적용되지 않습니다.

## 일정 설정 후 {#after-schedule-set}

저장되면 예약된 날짜와 시간이 기능 플래그 또는 기능 그룹 보기에 나타납니다.

일정이 활성화된 상태에서 수동으로 켜기/끄기 상태를 전환하려고 하면 일정을 재정의할지 작업을 취소할지 묻는 경고가 나타납니다.

## 예약된 날짜 및 시간에 {#on-schedule}

예약된 시간에 피쳐 플래그 또는 피쳐 그룹이 자동으로 활성화됩니다(ON 상태로 이동).

일정이 실행되면 **일정** 단추가 비활성화됩니다. 피쳐 플래그나 피쳐 그룹이 해제(off) 상태일 때만 스케줄을 설정할 수 있습니다.
