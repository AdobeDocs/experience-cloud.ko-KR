# 통합 단계 {#integration-steps}

애플리케이션 유형과 일치하는 통합 경로를 선택합니다.

## 웹 서비스 {#web-services}

백엔드 서비스는 서버측 SDK을 사용하여 통합됩니다. 특정 스택에 대한 지침은 경험 롤아웃 지원 센터에 문의하십시오.

>[!NOTE]
>
>서버측 SDK 설명서는 현재 준비 중이며 곧 제공될 예정입니다. 조기 액세스 지침은 Adobe 담당자에게 문의하십시오.

## 웹 및 모바일 애플리케이션 {#web-mobile}

웹 및 모바일 애플리케이션은 각각 웹 SDK 및 모바일 SDK을 통해 통합됩니다.

>[!NOTE]
>
>웹 SDK 및 모바일 SDK 설명서는 현재 준비 중이며 곧 제공될 예정입니다. 조기 액세스 지침은 Adobe 담당자에게 문의하십시오.

## 데스크탑 애플리케이션 {#desktop}

데스크탑 애플리케이션은 SDK을 통해 통합됩니다. 데스크탑 스택과 관련된 통합 지침은 Adobe 담당자에게 문의하십시오.

>[!NOTE]
>
>Desktop SDK 설명서는 현재 준비 중이며 곧 제공될 예정입니다.
