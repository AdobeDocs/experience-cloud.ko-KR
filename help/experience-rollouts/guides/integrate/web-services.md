# 웹 서비스 {#web-services}

백엔드 서비스는 서버측 SDK을 사용하여 Experience Rollouts와 통합됩니다. 기술 스택과 일치하는 SDK을 선택합니다.

## 사용 가능한 SDK {#available-sdks}

| 언어 | 통합 안내서 | 릴리스 정보 |
|---|---|---|
| **Java** | [Java SDK 통합 안내서](../sdk-releases/java/java-sdk-integration-guide.md) | [Java SDK 릴리스 노트](../sdk-releases/java/java-sdk-release-notes.md) |
| **Node.js** | [Node.js SDK 통합 안내서](../sdk-releases/nodejs/nodejs-sdk-integration-guide.md) | [Node.js SDK 릴리스 노트](../sdk-releases/nodejs/nodejs-sdk-release-notes.md) |

## 기타 언어 {#other-languages}

백엔드 서비스가 위에 나열되지 않은 언어를 사용하는 경우, Experience Rollouts 지원에 문의하여 특정 스택과 통합하는 방법에 대한 지침을 확인하십시오.
